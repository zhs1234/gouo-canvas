# 发布记录

## v0.1.0（2026-07-13）

首个光构（Gouo Canvas）产品化版本。

### 品牌与界面

- 完成“光构”中英文品牌、Logo、蓝色主题和仓库链接统一。
- 移除公开前台不再使用的 Agent 入口与用户 API 配置入口。
- 新增光构原生用户中心，普通用户无需进入 One Hub 面板。

### 账户与后端

- 接入基于 One Hub 的登录、注册、会话和统一 API 中继。
- 为每位用户生成独立中继令牌，平台上游 API Key 不下发到浏览器。
- 接入额度展示、兑换码充值和最近使用记录。

### 计费

- 图片生成、编辑与变体接口改为按成功请求次数固定计费。
- 默认价格为 ¥0.10/次，可通过 `GOUO_IMAGE_PRICE_CNY` 调整。
- 失败请求退回预扣额度，并增加固定价格计费测试。

### 文档与部署

- 完善本地开发、后端接入、Docker 部署和环境变量说明。
- 新增用户使用指南、常见问题和生产上线检查清单。
- 扩充 Git 忽略规则，避免提交环境变量、数据库、日志和私钥文件。
