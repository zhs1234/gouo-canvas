# 光构用户指南

[English](../en/user-guide.md) · [文档中心](../README.md)

本指南面向使用光构公开产品模式的创作者。管理员部署和渠道配置请阅读[部署总览](./deployment/index.md)和[后端接入说明](./backend.md)。

## 1. 注册与登录

1. 打开光构，选择“立即注册”。
2. 输入用户名和密码；界面显示的长度与验证码要求以当前服务端配置为准。
3. 注册成功后返回登录页，使用新账号进入工作区。
4. 顶栏“用户中心”可查看账户、余额、云端空间、兑换码和使用记录。

若运营方启用了邮箱服务，可在用户中心绑定邮箱，也可以通过登录页的“忘记密码”完成邮件验证和密码重置。服务未启用邮件时，相应入口不会生效，应联系管理员处理账号恢复。

登录状态由服务端 Cookie 保持。管理员更换 `SESSION_SECRET` 会使现有会话失效；更换 `USER_TOKEN_SECRET` 会使已有生成令牌失效。遇到“无效的令牌”时，先退出并重新登录。

## 2. 完成第一次生成

首次进入会显示三步引导，空画廊还提供示例提示词。

1. 在底部输入框描述主体、场景、构图、光线和限制条件。
2. 按需设置尺寸、质量、输出格式、透明背景、压缩率、审核方式和数量。
3. 可选：上传参考图，或为某张参考图绘制遮罩。
4. 检查发送按钮附近的预计费用，然后提交。
5. 任务完成后，打开图片进行下载、复制、收藏或继续编辑。

请求提交时会检查并预扣额度；服务端确认失败后会退回该次预扣。实际单价以用户中心和提交区显示为准，不要只依据文档中的示例价格。

## 3. 写提示词

清晰提示词通常包含：

- **主体**：画面中是谁或什么。
- **环境**：主体位于何处，周围有哪些元素。
- **表现形式**：摄影、插画、3D、海报、信息图等。
- **构图**：特写、全身、俯拍、广角、中心构图等。
- **光线与情绪**：柔光、逆光、霓虹、宁静、热烈等。
- **约束**：不要文字、不要水印、保留人物姿势或产品结构等。

示例：

```text
一个透明亚克力音乐播放器，摆放在深色金属桌面上。
高端产品摄影，中心构图，侧面柔光和蓝色边缘光，
背景简洁，不出现文字、品牌标志或水印。
```

提示词应优先描述可见结果。只写“更好看”“更专业”通常不如明确说明材质、视角、色彩和留白有效。

## 4. 使用灵感库

点击顶栏“灵感”打开灵感库。可以搜索或按摄影、人像、产品、海报、插画、空间和知识等分类浏览。

- 点击卡片查看完整提示词。
- “立即使用”会把提示词写入主页输入框，不会立即发送。
- 复制按钮只复制文本，不覆盖当前草稿。
- 使用示例后，建议替换主体、品牌限制、色彩和构图，使结果符合自己的用途。

## 5. 参考图与图片引用

可以通过文件选择、拖拽或粘贴添加图片，最多 16 张；实际请求还会受到上游渠道的数量、格式和总大小限制。

- 在提示词中说明每张图的用途，例如“保留图 1 的人物，采用图 2 的配色”。
- 输入 `@` 可选择当前参考图并插入明确引用。
- 桌面端可以拖动缩略图调整顺序；顺序变化时，已有图片引用会随之更新。
- 点击参考图的编辑入口，可根据设置选择替换图片或添加遮罩。
- 上传前确认自己有权处理图片，不要上传不希望交给平台或上游服务的数据。

参考图不会自动告诉模型“保留什么”。如果希望保留身份、姿势、产品轮廓或构图，必须写进提示词。

## 6. 局部遮罩编辑

1. 添加一张要修改的图片。
2. 点击该图片的编辑入口并选择“添加遮罩”。
3. 在遮罩编辑器中涂抹要改变的区域；蓝色区域代表待修改范围。
4. 保存遮罩，并在提示词中写明该区域要变成什么。
5. 提交编辑请求。

遮罩覆盖整图时，界面会提示这可能导致整张图片重绘。遮罩只是范围提示，不会替代文字编辑要求。

## 7. 查看和管理任务

任务卡片和详情页可提供以下操作，具体按钮取决于任务状态：

- 下载单张图片、全部输出或流式中间图。
- 复制图片或提示词。
- 重试失败任务。
- 复用任务参数并重新生成。
- 将输出作为参考图继续编辑。
- 添加到一个或多个收藏夹。
- 删除本地任务，或将已同步任务移入回收站。

顶部搜索支持提示词和参数关键词。状态筛选包括全部、已完成、生成中、失败、待同步、同步失败和回收站。桌面端支持框选及 `Ctrl`/`⌘` 多选，移动端支持多选操作；批量下载是否使用 ZIP 可在“设置 → 通用”中管理。

## 8. 收藏夹

- 同一个任务可以加入多个收藏夹。
- 收藏夹总览支持搜索、管理和批量下载。
- 删除收藏夹不会自动删除其中的任务。
- 登录模式下，收藏夹和任务关联会与账号一起同步。

## 9. 云端同步与回收站

登录后，任务、输出图、参考图、遮罩、缩略图和收藏夹会自动同步。浏览器 IndexedDB 同时保存缓存和待同步队列。

- 顶部同步条显示同步进度、错误和云端空间。
- 网络中断后，已缓存的作品仍可查看；恢复网络后系统会继续同步。
- 换浏览器后，登录同一账号可恢复已经同步的作品。
- 尚未同步成功的作品可能只存在原浏览器，清理站点数据前应先确认同步完成或导出备份。
- 已同步任务删除后进入回收站，仍占用云端空间，可在“回收站”筛选中恢复。
- 云端配额满时，新任务仍可能保存在本地，但会显示同步失败。

当前回收站是可恢复隐藏，不是物理文件清理。需要释放服务端空间时应联系管理员确认运营侧的清理策略。

## 10. 额度、兑换码与记录

用户中心提供：

- 当前余额和累计使用量。
- 当前单次图片请求价格。
- 云端空间使用量。
- 兑换码充值。
- 最近使用记录及 CSV 导出。

一次请求选择多张输出时，当前后端按一次成功图片请求计费。运营方可调整 `GOUO_IMAGE_PRICE_CNY`，因此界面显示价格是最终依据。若支付渠道尚未正式配置，只使用管理员提供的兑换码或人工充值方式。

## 11. 设置、导入与导出

公开产品模式隐藏 API Key 和服务商配置，普通用户不需要填写平台密钥。设置中仍可调整提交快捷键、提交后是否清空、是否保留输入草稿、参考图编辑行为、批量 ZIP 路径、重试按钮和系统通知等。

“设置 → 数据”可以导出或导入 ZIP 备份，也可以清理所选本地数据。即使账号启用了云端同步，执行清理或导入前仍应确认同步状态和备份内容，避免覆盖仅存在本地的数据。

## 12. 安装为应用

支持 PWA 的浏览器会在顶栏显示安装入口。iPhone/iPad 上可使用 Safari 的“分享 → 添加到主屏幕”。浏览器未提供安装事件时，界面会显示对应平台的手动步骤。

## 13. 常见问题

### 提示“无效的令牌”或 HTTP 401

退出后重新登录。如果持续出现，管理员可能更换了令牌密钥或重置了账号令牌。

### 余额不足

打开用户中心，使用有效兑换码或运营方已启用的充值方式补充额度，然后重试原任务。

### 请求被安全系统拒绝

删除可能涉及色情、暴力、违法、侵权或敏感身份的内容后重试。向管理员反馈时附上错误中的 request ID，但不要公开密码、Cookie 或完整令牌。

### 图片上传失败

尝试换用 PNG、JPEG 或 WebP，减少图片数量或文件大小。若使用遮罩，请确保遮罩主图仍在参考图列表中。

### 请求超时

降低尺寸、质量或输出数量后重试。若错误发生在接近固定时间点，管理员还应检查 CDN 和反向代理的超时限制。

### 生成成功但换设备后看不到

确认登录的是同一个账号，并查看原浏览器的同步状态。只有已经上传到云端的任务才能跨设备恢复。

### 页面没有 API 配置

这是公开产品模式的预期行为。平台统一管理上游 API，普通用户不应接触平台 API Key。开发者需要直接测试 API 时，请使用[开发指南](./development.md)中的纯前端模式。

## 14. 数据与隐私

生产模式下，账号、额度、使用记录和已同步作品由服务端保存，浏览器保存缓存和待同步数据。生成或编辑时，提示词与必要图片会发送给光构后端以及管理员配置的上游模型服务。运营方应公开用户协议、隐私政策、内容规则、数据保留周期、退款说明和侵权投诉渠道。
