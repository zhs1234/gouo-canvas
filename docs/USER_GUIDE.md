# 光构用户使用指南

光构（Gouo Canvas）是一个 AI 图片生成与编辑平台。你可以从文字开始创作，也可以上传参考图继续修改。

## 注册与登录

1. 打开光构，选择注册账号。
2. 使用用户名和密码完成注册并登录。
3. 登录后，右上角用户入口可查看当前额度、兑换码充值和最近使用记录。

登录状态由服务端会话保持。如果服务端更换了会话密钥、令牌密钥或清理了账户数据，系统可能要求重新登录。

## 快速开始

首次进入光构时会显示三步引导，介绍提示词、参考图、遮罩、计费和作品保存方式。空画廊还会提供 6 个示例提示词，点击即可自动填入底部输入框。

1. 在底部输入框描述想生成的画面。
2. 按需要选择尺寸、质量、格式、透明背景、审核方式和数量。
3. 可选：点击附件按钮上传参考图。
4. 提交后等待生成，结果会自动进入画廊。
5. 打开满意的图片，可下载、收藏或继续作为参考图编辑。

生成请求会消耗账户额度。余额不足时，请在用户中心使用兑换码或管理员已启用的充值方式补充额度。

提交按钮上方会显示本次请求的预计扣费；当前固定为成功请求 ¥0.10/次，失败请求自动退回。第一次生成完成后，页面会引导下载、收藏或将输出作为参考图继续编辑。

## 使用灵感库

点击顶栏的“灵感”进入灵感库。可以按摄影、人像、产品、海报、插画、空间和知识分类浏览，也可以搜索风格、场景或用途。

- 点击卡片查看完整提示词。
- 点击“立即使用”会把提示词写入主页输入框，之后仍可自由修改主体、色彩和构图。
- 点击复制按钮只复制提示词，不改变当前输入。

灵感库的产品结构参考了 [awesome-gpt-image-2](https://github.com/freestylefly/awesome-gpt-image-2)、[awesome-gpt-image](https://github.com/ZeroLu/awesome-gpt-image) 和 [stable-diffusion-image-prompt-gallery](https://github.com/joelparkerhenderson/stable-diffusion-image-prompt-gallery)。内置中文提示词由光构重新编写，没有直接收录社区案例图片或来源不明确的原始提示词。

## 如何写出更好的提示词

一个清晰的提示词通常包含：

- 主体：画面中是谁或什么。
- 环境：主体位于哪里。
- 风格：摄影、插画、3D、平面设计等。
- 构图：近景、全身、俯拍、广角或特写。
- 光线与情绪：柔光、霓虹、电影感、宁静或热烈。
- 约束：不要文字、不要水印、保留人物姿势等。

示例：

```text
一个透明亚克力音乐播放器，摆放在深色金属桌面上，
产品摄影，侧面柔光，蓝色边缘光，
中心构图，高级、简洁，没有文字，没有水印。
```

## 参考图与编辑

- 可通过文件选择、拖拽或粘贴上传参考图。
- 第一次上传参考图时，光构会提示如何在提示词中说明每张图片的用途。
- 一次最多可使用 16 张参考图，实际上游限制以管理员配置为准。
- 提示词中应说明每张图的用途，例如“保留图 1 的人物，使用图 2 的服装风格”。
- 如只想修改局部，可使用遮罩编辑器涂抹需要更改的区域。
- 第一次打开遮罩编辑器时会显示简短图示；蓝色画笔经过的区域是需要修改的区域，保存后仍需在提示词中说明希望改成什么。
- 上传前请确认你有权处理图片，不要上传不希望交给平台或上游服务的数据。

## 画廊和收藏夹

- 历史任务会显示在画廊中，可按状态和关键词筛选。
- 同一个任务可放入多个收藏夹。
- 支持批量选择、下载、导出和删除。
- 登录后，画廊任务、图片、参考图、遮罩和收藏夹会自动同步到云端；第一次升级会自动迁移当前浏览器中的历史作品。
- 浏览器仍会保留本地缓存。网络中断时可继续查看已缓存作品，恢复网络后会自动续传。
- 每个用户默认拥有 2 GB 云端空间；空间不足时新作品仍保留在当前浏览器，但会标记为“同步失败”。
- 删除已经同步的作品会将其移入回收站，图片文件仍会保留并继续占用空间，可从“回收站”筛选中恢复。

## 额度与计费

当前图片生成、编辑和变体接口按成功请求次数固定计费，默认价格为 **¥0.10/次**，实际价格以运营方在后端设置的 `GOUO_IMAGE_PRICE_CNY` 为准。

- 一次请求选择多张输出时，当前仍按一次成功请求计费。
- 请求提交时会检查并预扣额度。
- 平台确认请求失败时会退回本次预扣额度。
- 用户中心可查看余额和最近使用记录。
- 上游内容安全策略拒绝请求时属于生成失败，具体提示中会包含可供排查的 request ID。

## 常见问题

### 提示“无效的令牌”

先退出账号并重新登录。如果问题持续出现，可能是服务端更换了 `USER_TOKEN_SECRET` 或管理员重置了用户令牌，需要联系管理员处理。

### 请求被安全系统拒绝

修改提示词，移除可能涉及色情、暴力、违法或侵权的描述后重试。这是上游安全策略的拒绝，不代表光构前端故障。向管理员反馈时请附上错误中的 request ID，但不要公开账号密码或完整令牌。

光构会在常见错误后追加“下一步”建议。余额或额度不足时会直接打开用户中心兑换区；网络错误、安全审核、令牌失效和图片格式问题也会给出对应处理方法。

### 页面中没有 API 配置

公开产品模式由平台统一提供上游 API，普通用户无需也不能在前端填写平台 API Key。

### 生成成功但画廊不见了

确认是否登录了生成作品时使用的同一个光构账号，并检查顶部同步状态。换浏览器或清理站点数据后，登录同一账号会从云端恢复已经同步的作品；尚未完成同步的本地作品只能在原浏览器中找到。

## 数据与隐私

账户、额度、调用记录和已同步的作品资产由服务端保存；浏览器 IndexedDB 保存本地缓存和待同步数据。图片生成时，提示词与必要的参考图会发送至光构后端和管理员配置的上游模型服务。公开运营前，运营方应发布正式的用户协议、隐私政策、内容规则和退款说明。
