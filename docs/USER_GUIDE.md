# 光构用户使用指南

光构（Gouo Canvas）是一个 AI 图片生成与编辑平台。你可以从一段文字开始创作，也可以上传参考图继续修改。

## 快速开始

1. 登录光构账号。
2. 进入“画廊”，在底部输入框描述想要的画面。
3. 选择画面尺寸、质量和生成数量。
4. 提交后等待生成，结果会自动进入画廊。
5. 打开满意的图片，可下载、收藏或继续作为参考图编辑。

> 注：登录和平台额度能力属于 SaaS 上线阶段。当前开发版仍保留原项目的本地 API 配置界面，用于功能验证。

## 如何写出更好的提示词

一个清晰的提示词通常包含：

- 主体：画面中是谁或什么。
- 环境：主体位于哪里。
- 风格：摄影、插画、3D、平面设计等。
- 构图：近景、全身、俯拍、广角或特写。
- 光线与情绪：柔光、霓虹、电影感、宁静或热烈。
- 约束：不要文字、不要水印、保留人物姿势等。

示例：

```text
一个透明亚克力音乐播放器，摆放在深色金属桌面上，
产品摄影，侧面柔光，青色和靛蓝色边缘光，
中心构图，高级、简洁，没有文字，没有水印。
```

## 参考图与编辑

- 可通过文件选择、拖拽或粘贴上传参考图。
- 一次最多可使用 16 张参考图。
- 提示词中应说明每张图的用途，例如“保留图 1 的人物，使用图 2 的服装风格”。
- 如只想修改局部，可使用遮罩编辑器涂抹需要更改的区域。

## Agent 创作模式

Agent 模式适合多轮创作。你可以先让 Agent 生成初稿，再说“保留构图，把灯光改成黄昏”。输入 `@` 可引用已上传或前面轮次生成的图片。

## 画廊和收藏夹

- 历史任务会显示在画廊中。
- 可按生成状态和关键词筛选任务。
- 同一个任务可放入多个收藏夹。
- 支持批量选择、下载和删除。

## 额度和扣费规则

正式版中，每次生成会根据模型、分辨率、质量和图片数量预估所需额度。

- 提交任务时先预扣额度。
- 任务成功后按实际消耗结算。
- 任务因平台或上游服务异常失败时，预扣额度应自动退回。
- 额度明细中可查看充值、消耗、退回和管理员调整记录。

## 数据与隐私

当前开发版的画廊记录和图片主要保存在浏览器 IndexedDB 中。SaaS 版上线后，需要在隐私政策中明确服务端保存的账户、订单、任务和图片数据，以及数据保留时间。
